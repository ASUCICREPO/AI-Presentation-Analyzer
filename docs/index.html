✦ Features

Everything You Need to Level Up

From real-time transcription to AI-driven analytics, every tool you need to become a better presenter — in one place.

🎙️

Live Voice Q&A

Engage in a bidirectional voice conversation with an AI interviewer powered by Amazon Nova Sonic for natural speech-to-speech interaction.

📝

Real-Time Transcription

Stream live transcription of your presentation using Amazon Transcribe, capturing every word as you speak for accurate analysis.

🎭

Configurable AI Personas

Choose or create custom interviewer personas with unique personalities, voices, and time limits to simulate different audience types.

📊

Post-Session Analytics

Receive detailed presentation analytics generated by Amazon Bedrock, including delivery feedback, content analysis, and improvement suggestions.

🔒

Content Safety Guardrails

Bedrock Guardrails screen all user-submitted persona text for harmful content and prompt injection attacks before it's persisted.

☁️

Fully Serverless

Built entirely on AWS serverless services — Lambda, DynamoDB, S3, Cognito, and Bedrock AgentCore — for zero infrastructure management.

Four Steps to Better Presentations

A streamlined workflow that takes you from upload to actionable insights in minutes.

Upload or Record

Upload a presentation video or PDF directly to S3 using secure presigned URLs, or record yourself live in the browser.

Practice Session

Deliver your presentation while real-time transcription captures your speech. Review the live transcript as you go.

AI Voice Q&A

Enter a live voice Q&A with a configurable AI persona that asks challenging questions about your presentation content.

Review Analytics

Get comprehensive post-session analytics with delivery scores, content feedback, and personalized improvement recommendations.

🚀 Deploy

Get Up and Running

Two deployment paths — pick the one that fits your workflow.

⚡ Automated Deploy

Quick setup via AWS CloudShell — 15 to 25 minutes

The script creates a CodeBuild project and deploys all four CDK stacks automatically. Works in any bash terminal with AWS CLI configured.

# Clone and deploy
git clone https://github.com/ASUCICREPO/AI-Presentation-Analyzer.git
cd AI-Presentation-Analyzer
chmod +x deploy.sh
./deploy.sh

🔧 Manual CDK Deploy

Full local control — requires Node.js 18+, CDK, Docker, Python 3.13

For developers who want granular control over each stack. Deploy individual stacks, customize parameters, and iterate locally.

# Install and deploy
cd backend
npm install
npx cdk bootstrap
npx cdk deploy --all
